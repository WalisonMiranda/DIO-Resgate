#DIO-Resgate

Jogo de nave para navegador desenvolvido no bootcamp Órbi Web Games Developer na plataforma digital innovation one.


## Funcionalidades

- Movimentar o helicóptero para cima e para baixo
- atirar



## Stack utilizada

**HTML** **CSS** **JavaScript** e **jQuery**
